# Tanker

A set of usefull services for the https://github.com/o-shabashov/cassie project:

* PostgreSQL
* Redis
* Redis Insight
* Memcached
* Minio
* Typesense
* Typesense dashboard
* Meilisearch
* Meilisearch dashboard
* Nginx-proxy
* Searxng

## Install
```bash
git clone https://github.com/o-shabashov/tanker
cd tanker
docker compose up -d
```
